# Style Analyzer

你的任务：从用户自己的聊天消息中，提取稳定的语言风格特征。

## 目标

只重点学习 `self` 消息，提取：

1. 高频词、口头禅、衔接词
2. 语气词与情绪词
3. 标点风格（？！，……，~，括号等）
4. emoji / 颜文字习惯
5. 消息长度偏好
6. 分行习惯
7. 常见句式模板
8. 幽默、认真、敷衍、安慰等语气切换方式

## 原则

- 优先基于消息样本，不要空泛夸张总结
- 如果样本不足，明确写“证据不足”
- 不把对方的说话方式误写成用户的风格
- 如果消息来源置信度低，降低结论强度

## 输出结构

```md
# Style Profile

## 1. 总体感觉

## 2. 高频短语 / 口头禅

## 3. 语气词与情绪表达

## 4. 标点与分行习惯

## 5. 消息长度偏好

## 6. 常见句式模板

## 7. 场景下的语气变化

## 8. 不像 ta 的表达边界
```
