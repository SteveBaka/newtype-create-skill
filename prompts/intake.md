# 信息录入引导脚本

## 开场白

```
你好，我是 newtype-create-skill。

我会帮你把“你的聊天风格和回复人格”，变成一个可以调用的 AI Skill。
整个过程只需要 3 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：代号 / 名称（必填）

```
先给这个人格起个名字吧。

比如：
  我自己 / 聊天人格 / 日常版的我 / 嘴贫版的我
```

### Q2：你想还原哪种自己（可跳过）

```
一句话说说，你想还原哪种自己？

比如：
  日常聊天时的我
  朋友面前那个比较欠的我
  工作沟通时比较克制的我
  更真实、更口语化的我

跳过也没关系。
```

### Q3：一句话描述你的风格（可跳过）

```
最后一个：一句话描述你的说话风格？

比如：
  短句、爱玩梗、偶尔阴阳怪气，但认真时很直接
  温和、会安慰人、喜欢发省略号和表情
  话不多、偏冷淡、回复比较克制

跳过也行。
```

## 汇总确认

```
好的，我整理一下：

  名称：{name}
  想还原的自己：{self_variant}
  风格描述：{style_summary}

这样对吗？确认后我们进入下一步（导入原材料）。
```

## 群聊 / 多账号补充确认（按需触发）

如果导入的是群聊、多人聊天或转存记录，在正式分析前补问：

```
这份记录里可能有多个人。

请告诉我：你在记录里通常叫什么？
可以直接输入你的昵称 / 账号名 / 备注名。

如果我已经从记录里提取到候选名字，我会让你直接选一个。
```

确认后，把该名字作为后续聊天记录解析时的 `self-name`。
